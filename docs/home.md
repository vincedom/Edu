Ce projet a pour but de proposer un agent conversationnel à destination des enfants sous la supervision des parents leur permettant de faire leur travail notamment leur devoir en autonomie partielle. 

Autrement dit il permet de délégue les tâches simples et rébarbatives de l'accompagnement de l'enfant dans ces devoirs. 

Il est construit autour d'un agent conversationnel audio, avec possibilité de prise de photo:
- l'enfant ne doit pas pouvoir sortir de l'application (mode kiosk) 
- un suivi pédagogique est assuré.
- l'agent dispose de skills notamment :
  - notification des parents en cas d'attention à apporter dans le domaine du travail ou du comportement 
  - notion sur les sciences cognitives et les mécanismes de mémorisation 
  - base de connaissances / exercices 
  - affichage d'image pour une meilleure compréhension de l'enfant.
